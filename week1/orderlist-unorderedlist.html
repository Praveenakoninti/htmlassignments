HTML (HyperText Markup Language)
HTML is the standard markup language used for creating the structure and content of webpages.
CSS (Cascading Style Sheets)
CSS is used for describing the presentation and styling of webpages, including layout, colors, fonts, and more.
JavaScript
JavaScript is a programming language that enables interactivity and dynamic behavior on webpages, allowing for client-side scripting.
Node.js
Node.js is a server-side JavaScript runtime environment that allows developers to run JavaScript on the server, enabling backend development.
Express.js
Express.js is a minimalistic and flexible web application framework for Node.js, used to build server-side web applications and APIs.
React
React is a JavaScript library for building user interfaces. It is commonly used for creating interactive and reusable frontend components.
Angular
Angular is a TypeScript-based web application framework maintained by Google. It is used for building robust and scalable web applications.
Vue.js
Vue.js is a progressive JavaScript framework used for building user interfaces. It focuses on simplicity and ease of integration.
MySQL
MySQL is an open-source relational database management system used for storing and managing structured data in web applications.
MongoDB
MongoDB is a NoSQL database that provides a flexible and scalable approach to storing and retrieving data, commonly used in modern web development.
